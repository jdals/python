03-1 if 문
==================
* 프로그래밍에서 조건을 판단하여 해당 조건에 맞는 상황을 수행하는데 쓰는 것.
* if 문을 만들 때는 if 조건문: 바로 다음 문장부터 if 문에 속하는 모든 문장에 들여쓰기를 해야함.
* 파이썬은 들여쓰기를 하지 않으면 오류가 발생함.

##### if 문의 기본 구조
* 조건문이 참이면 if 블록들을 수행함. 조건문이 거짓이면 else 블록들을 수행함.
* else문은 if 문 없이 독립적으로 사용할 수 없음.
<pre>
<code>
  if 조건문:
      수행할_문장 1
      수행할_문장 2
  else:
      수행할_문장 1
      수행할_문장 2
</code>
</pre>
