03-3 for 문
==================
* 문장을 반복해서 수행해야 할 경우 사용함.

##### for 문의 기본 구조
* 리스트나 튜플, 문자열의 첫 번째 요소부터 차례로 변수로 대입되어 수행됨.
<pre>
<code>
  for 변수 in 리스트(또는 튜플, 문자열):
      수행할_문장1
      수행할_문장2
</code>
</pre>

* 전형적인 for 문
<pre>
<code>
  test_list = ['one', 'two', 'three']
  for i in test_list:
      print(i)
</code>
</pre>

* 다양한 for 문의 사용
<pre>
<code>
  a = [(1, 2), (3, 4), (5, 6)]
  for (first, last) in a:
      print(first + last)
</code>
</pre>

##### for 문과 continue 문
<pre>
<code>
  marks = [90, 25, 67, 45, 80]
  number = 0
  for mark in marks:
      number = number + 1
      if mark < 60:
          continue
      print("%d번 학생 축하합니다. 합격입니다. " % number)
</code>
</pre>

##### range 함수
* 숫자 리스트를 자동으로 만들어주는 함수. for문과 함께 사용하는 경우가 많음.
<pre>
<code>
  a = range(10)
  a  ->  range(0, 10), 0부터 10 미만의 숫자를 포함하는 range 객체.
  b = range(1, 11)  ->  시작, 끝 지정 가능. 끝 숫자는 포함하지 않음.
</code>
</pre>

<pre>
<code>
  add = 0
  for i in range(1, 11):
      add = add + 1
  print(add)  ->  55
</code>
</pre>

* for, range 를 이용한 구구단
<pre>
<code>
  for i in range(2, 10):
      for j in range(1, 10):
          print(i*j, end=" ")
      print('')
</code>
</pre>

##### 리스트 컴프리헨션
* 리스트 안에 for 문을 포함하는 방식.
<pre>
<code>
  [표현식 for 항목 in 반복_가능_객체 if 조건문]

  [표현식 for 항목1 in 반복_가능_객체1 if 조건문1
         for 항목2 in 반복_가능_객체2 if 조건문2]
<pre>
<code>
  a = [1, 2, 3, 4]
  result = []
  for num in a:
      result.append(num*3)
  print(result)  ->  [3, 6, 9, 12]


  a = [1, 2, 3, 4]
  result = [num*3 for num in a]
  print(result)  ->  [3, 6, 9, 12]

  a = [1, 2, 3, 4]
  result = [num*3 for num in a if num%2 == 0]
  print(result)  ->  [6, 12]
</code>
</pre>
<pre>
<code>
  result = [x*y for x in range(2, 10)
                for y in range(1, 10)]
  print(result)  ->  [2, 4, 6, 8 ... 63, 72, 81]
</code>
</pre>
