03-1 if 문
==================
* 프로그래밍에서 조건을 판단하여 해당 조건에 맞는 상황을 수행하는데 쓰는 것.
* if 문을 만들 때는 if 조건문: 바로 다음 문장부터 if 문에 속하는 모든 문장에 들여쓰기를 해야함.
* 파이썬은 들여쓰기를 하지 않으면 오류가 발생함.

##### if 문의 기본 구조
* 조건문이 참이면 if 블록들을 수행함. 조건문이 거짓이면 else 블록들을 수행함.
* else문은 if 문 없이 독립적으로 사용할 수 없음.
<pre>
<code>
  if 조건문:
      수행할_문장 1
      수행할_문장 2
  else:
      수행할_문장 1
      수행할_문장 2
</code>
</pre>

##### 조건문이란
* 참과 거짓을 판단하는 문장임.
* 비교 연산자
<pre>
<code>
  x < y  ->  x가 y보다 작음.
  x > y  ->  x가 y보다 큼.
  x == y  ->  x와 y가 같음.
  x != y  ->  x와 y가 같지 않음.
  x >= y  ->  x가 y보다 크거나 같음.
  x <= y  ->  x가 y보다 작거나 같음.
</code>
</pre>

<pre>
<code>
  x = 3
  y = 2
  x > y  ->  True
  x < y  ->  False
  x == y  ->  False
  x != y  ->  True
</code>
</pre>

* and, or, not
<pre>
<code>
  x or y  ->  x와 y 둘 중 하나만 참이어도 참임.
  x and y  ->  x와 y 모두 참이어야 참임.
  not x  ->  x가 거짓이면 참임.
</code>
</pre>

* in, not in  ->  x in, not in 리스트, 튜플, 문자열
<pre>
<code>
  1 in [1, 2, 3]  ->  True
  1 not in [1, 2, 3]  ->  False
  'a' in ('a', 'b', 'c')  -> True
  'j' not in 'python'  ->  True
</code>
</pre>

* pass  ->  조건문에서 참, 거짓에 따라 실행할 행동을 정의할 때 아무것도 하지 않도록 설정함.

##### 다양한 조건을 판단하는 elif
* elif는 이전 조건문이 거짓일 떄 수행됨.(개수에 제한 없음)
<pre>
<code>
  if 조건문:
      수행할_문장1
      수행할_문장2
  elif 조건문:
      수행할_문장1
      수행할_문장2
  else:
      수행할_문장1
      수행할_문장2
</code>
</pre>

##### 조건부 표현식
* 변수 = 조건문이 참인 경우의 값 if 조건문 else 조건문이 거짓인 경우의 값
<pre>
<code>
  if score >= 60:
      message = "success"
  else:
      message = "failure"

  message = "success" if score >= 60 else "failure"
</code>
</pre>
