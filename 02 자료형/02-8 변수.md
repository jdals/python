02-8 자료형의 값을 저장하는 공간, 변수
==================
* 변수를 만들 때는 = (assignment) 기호를 사용함.
* 변수_이름 = 변수에_저장할_값
* 파이썬은 변수에 저장된 값을 스스로 판단하여 자료형의 타입을 지정함.

##### 변수 만들기
* a, b, c를 변수라고 함.
<pre>
<code>
  a = 1
  b = "python"
  c = [1, 2, 3]
</code>
</pre>

##### 변수란?
* 파이썬에서의 객체를 가리킴. 객체는 자료형의 데이터(값)와 같은 것.
* [1, 2, 3] 값을 가지는 리스트 데이터(객체)가 자동으로 메모리에 생성됨.
* 변수 a는 [1, 2, 3] 리스트가 저장된 메모리의 주소를 가리킴.
<pre>
<code>
  a = [1, 2, 3]
  id(a)  ->  4303029896  ->  id는 변수가 가리키고 있는 객체의 주소 값을 리턴.
</code>
</pre>

##### 리스트 복사
* b 변수에 a 변수를 대입하면 b와 a는 완전히 동일해짐.
* 해당 리스트 객체를 참조하는 변수가 a에서 a, b 두개로 늘어나는 것뿐임.
<pre>
<code>
  a = [1, 2, 3]
  b = a
  id(a)  ->  4303029896
  id(b)  ->  4303029896
  a is b  ->  Ture  ->  동일한 객체를 가리키는지 여부를 판단하는 명령어 is
</code>
</pre>

* a 리스트의 요소 값을 바꾸면 b도 똑같이 바뀌게 됨.
<pre>
<code>
  a[1] = 4
  a  ->  [1, 4, 3]
  b  ->  [1, 4, 3]
</code>
</pre>

* b 변수를 생성할 때 a 변수의 값을 가져오면서 다른 주소를 가리키게 하는 2가지 방법.

* [:] 이용하기  ->  리스트 전체를 가리킴.
<pre>
<code>
  a = [1, 2, 3]
  b = a[:]
  a[1] = 4
  a  ->  [1, 4, 3]
  b  ->  [1, 2, 3]
</code>
</pre>

* copy 모듈 이용하기
<pre>
<code>
  from copy imort copy  ->  copy 모듈에 있는 copy 함수 import
  a = [1, 2, 3]
  b = copy(a)  ->  b = a[:]와 동일한 기능.
  b is a  ->  False
</code>
</pre>

##### 변수를 만드는 여러 가지 방법
* 튜플로 a, b 에 값을 대입할 수 있음. 튜플은 괄호 생략 가능.
<pre>
<code>
  a, b = ('python', 'life')
  (a, b) = 'python', 'life'
</code>
</pre>

* 리스트로 변수 만들기도 가능.
<pre>
<code>
  [a, b] = ['python', 'life']
</code>
</pre>

* 여러 개의 변수에 같은 값 대입 가능.
<pre>
<code>
  a = b = 'python'
</code>
</pre>

* 파이썬에서는 두 변수의 값을 매우 간단하게 바꿀 수 있음.
<pre>
<code>
  a = 3
  b = 5
  a, b = b, a  ->  a, b 값을 바꿈.
  a  ->  5
  b  ->  3
</code>
</pre>
