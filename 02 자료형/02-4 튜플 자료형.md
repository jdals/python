02-4 튜플 자료형
==================
* 튜플은 리스트와 거의 비슷함.
* 리스트는 [], 튜플은 () 으로 둘러쌈.
* 리스트는 요솟값의 생성, 삭제, 수정이 가능하지만, 튜플은 요솟값을 바꿀 수 없음.
* 튜플 요솟값은 변경하려고 하면 오류가 발생함.

##### 튜플 만들기
<pre>
<code>
  t1 = ()
  t2 = (1,)  ->  요소 한개만 가질때는 쉼표를 붙여야함.
  t3 = (1, 2, 3)
  t4 = 1, 2, 3  ->  소괄호 생략 가능.
  t5 = ('a', 'b', ('ab', 'cd'))
</code>
</pre>

##### 튜플 다루기
* 튜플 인덱싱
<pre>
<code>
  t1 = (1, 2, 'a', 'b')
  t1[0]  ->  1
  t1[3]  ->  'b'
</code>
</pre>

* 튜플 슬라이싱
<pre>
<code>
  t1 = (1, 2, 'a', 'b')
  t1[1:]  ->  (2, 'a', 'b')
</code>
</pre>

* 튜플 더하기
<pre>
<code>
  t1 = (1, 2, 'a', 'b')
  t2 = (3, 4)
  t3 = t1 + t2
  t3  ->  (1, 2, 'a', 'b', 3, 4)
</code>
</pre>

* 튜플 곱하기
<pre>
<code>
  t2 = (3, 4)
  t3 = t2 * 3
  t3  ->  (3, 4, 3, 4, 3, 4)
</code>
</pre>

* 튜플 길이 구하기
<pre>
<code>
  t1 = (1, 2, 'a', 'b')
  len(t1)  ->  4
</code>
</pre>
