02-3 리스트 자료형
==================
##### 리스트 만들기
<pre>
<code>
  리스트명 = [요소1, 요소2, 요소3, ...]
</code>
</pre>

* 리스트 안에는 어떠한 자료형도 포함할 수 있음.
<pre>
<code>
  a = []
  b = [1, 2, 3]
  c = ['Life', 'is', 'too', 'short']
  d = [1, 2, 'Life', 'is']
  e = [1, 2, ['Life', 'is']]
</code>
</pre>

##### 리스트 인덱싱
<pre>
<code>
  a = [1, 2, 3]
  a[0]  ->  1
  a[0] + a[2]  ->  4
  a[-1]  ->  3
</code>
</pre>
<pre>
<code>
  a = [1, 2, 3, ['a', 'b', 'c']]
  a[0]  ->  1
  a[-1]  ->  ['a', 'b', 'c']
  a[3]  ->  ['a', 'b', 'c']
  a[-1][0]  ->  'a'
</code>
</pre>

##### 리스트 슬라이싱
<pre>
<code>
  a = [1, 2, 3, 4, 5]  ->  리스트
  a[0:2]  ->  [1, 2]
  b = "12345"  ->  문자열
  b[0:2]  ->  '12'
</code>
</pre>
<pre>
<code>
  a = [1, 2, 3, 4, 5]
  b = a[:2]  ->  [1, 2]
  c = a[2:]  ->  [3, 4, 5]
</code>
</pre>

##### 리스트 연산
* 리스트 더하기(+)
<pre>
<code>
  a = [1, 2, 3]
  b = [4, 5, 6]
  a + b  ->  [1, 2, 3, 4, 5, 6]
</code>
</pre>

* 리스트 반복하기(*)
<pre>
<code>
  a = [1, 2, 3]
  a * 3  ->  [1, 2, 3, 1, 2, 3, 1, 2, 3]
</code>
</pre>

* 리스트 길이 구하기
<pre>
<code>
  a = [1, 2, 3]
  len(a)  ->  3
</code>
</pre>
<pre>
<code>
  a = [1, 2, 3]
  a[2] + "hi"  ->  error
  str(a[2]) + "hi"  ->  3hi
</code>
</pre>

##### 리스트 요소 수정
<pre>
<code>
  a = [1, 2, 3]
  a[2] = 4
  a  ->  [1, 2, 4]
</code>
</pre>

##### 리스트 요소 삭제
<pre>
<code>
  a = [1, 2, 3]
  del a[1]  ->  del 객체  형식으로 모든 자료형에 사용 가능한 삭제 함수임.
  a  ->  [1, 3]
</code>
</pre>
<pre>
<code>
  a = [1, 2, 3, 4, 5]
  del a[2:] 
  a  ->  [1, 2]
</code>
</pre>

##### 리스트 관련 함수
* 리스트 요소 추가하기  ->  append(x)
<pre>
<code>
  a = [1, 2, 3]
  a.append(4)
  a  ->  [1, 2, 3, 4]
  a.append([5, 6])
  a  ->  [1, 2, 3, 4, [5, 6]]
</code>
</pre>

* 리스트 정렬  ->  sort()
<pre>
<code>
  a = [1, 4, 3, 2]
  a.sort()
  a  ->  [1, 2, 3, 4]
</code>
</pre>

* 리스트 뒤집기  ->  reverse()  ->  리스트 그대로 뒤집음.
<pre>
<code>
  a = ['a', 'c', 'b']
  a.reverse()
  a  ->  ['b', 'c', 'a']  
</code>
</pre>

* 인덱스 반환  ->  index(x)  ->  x의 값이 리스트에 있으면 인덱스 값 반환
<pre>
<code>
  a = [1, 2, 3]
  a.index(3)  ->  2
  a.index(1)  ->  0
  a.index(0)  ->  error
</code>
</pre>

* 리스트 요소 삽입  ->  insert(a, b)  ->  a번째 위치에 b 삽입
<pre>
<code>
  a = [1, 2, 3]
  a.insert(0, 4)
  a  ->  [4, 1, 2, 3]
  a.insert(3, 5)
  a  ->  [4, 1, 2, 5, 3]
</code>
</pre>

* 리스트 요소 제거  ->  remove(x)  ->  리스트에서 첫 번째로 나오는 x 삭제
<pre>
<code>
  a = [1, 2, 3, 1, 2, 3]
  a.remove(3)
  a  ->  [1, 2, 1, 2, 3]
  a.remove(3)
  a  ->  [1, 2, 1, 2]
</code>
</pre>

* 리스트 요소 끄집어 내기  ->  pop()  ->  리스트 맨 마지막 요소 리턴하고 삭제
<pre>
<code>
  a = [1, 2, 3]
  a.pop()  ->  3
  a  ->  [1, 2]
</code>
</pre>
* pop(x)는 x번째 요소를 리턴하고 삭제
<pre>
<code>
  a = [1, 2, 3]
  a.pop(1)  ->  2
  a  ->  [1, 3]
</code>
</pre>

* 리스트에 포함된 요소 x의 개수 세기  ->  count(x)  ->  리스트 안의 x 개수 리턴
<pre>
<code>
  a = [1, 2, 3, 1]
  a.count(1)  ->  2
</code>
</pre>

* 리스트 확장  -> extend(x)  -> x는 리스트만 가능. 원래 리스트에 x 리스트를 더함.
<pre>
<code>
  a = [1, 2, 3]
  a.extend([4, 5])
  a  ->  [1, 2, 3, 4, 5]
  b = [6, 7]
  a.extend(b)
  a  ->  [1, 2, 3, 4, 5, 6, 7]

  "a.extend([4, 5])는 a += [4, 5], a = a + [4, 5] 와 동일함."
</code>
</pre>
