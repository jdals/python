02-7 불 자료형
==================
* 참과 거짓을 나타내는 자료형.
* True(참), False(거짓)의 두 가지 값만 가질 수 있음.

##### 불 자료형 사용
* 따옴표로 감싸지 않은 문자열을 변수에 지정해도 오류가 발생하지 않음.
<pre>
<code>
  a = True
  b = False
  
  type(a)  ->  <class 'bool'>
  type(b)  ->  <class 'bool'>
</code>
</pre>

* 조건문의 리턴값
<pre>
<code>
  1 == 1  ->  True
  2 > 1  ->  True
  2 < 1  ->  False
</code>
</pre>

##### 자료형의 참과 거짓
* 문자열, 리스트, 튜플, 딕셔너리 등의 값이 비어 있으면 거짓.("", [], (), {}
* 비어 있지 않으면 참이 됨.
* 숫자는 값이 0이면 거짓.
* None은 거짓을 의미함.
<pre>
<code>
  a = [1, 2, 3, 4]
  while a:  ->  a가 참인 동안
      print(a.pop())  ->  리스트의 마지막 요소를 하나씩 꺼냄.
</code>
</pre>

##### 불 연산
<pre>
<code>
  bool('python')  ->  True
  bool('')  ->  False
</code>
</pre>
