02-6 집합 자료형
==================
* 집합(set)은 집합에 관련된 것을 쉽게 처리하기 위해 만든 자료형임.
* 중복을 허용하지 않음. 이때문에 중복 제거를 위한 필터로 종종 사용됨.
* 순서가 없음(Unordered)

##### 집합 만들기
* set 키워드 사용
<pre>
<code>
  s1 = set([1, 2, 3])
  s1  ->  {1, 2, 3}
  s2 = set("Hello")
  s2  ->  {'e', 'H', 'l', 'o'}
</code>
</pre>

* 집합 자료형을 인덱싱으로 접근하려면 리스트나 튜플로 변환 후 접근해야함.
<pre>
<code>
  s1 = set([1, 2, 3])
  l1 = list(s1)
  l1[0]  ->  1
  t1 = tuple(s1)
  t1[0]  ->  1
</code>
</pre>

##### 교집합, 합집합, 차집합 구하기
<pre>
<code>
  s1 = set([1, 2, 3, 4, 5, 6])
  s2 = set([4, 5, 6, 7, 8, 9])
</code>
</pre>

* 교집합 구하기 (2가지)
<pre>
<code>
  s1 & s2  ->  {4, 5, 6}
  
  s1. intersection(s2)  ->  {4, 5, 6}
</code>
</pre>

* 합집합 구하기 (2가지)  ->  중복값은 1개씩만 표현됨.
<pre>
<code>
  s1 | s2  ->  {1, 2, 3, 4, 5, 6, 7, 8, 9}

  s1.union(s2)  ->  {1, 2, 3, 4, 5, 6, 7, 8, 9}
</code>
</pre>

* 차집합 구하기 (2가지)
<pre>
<code>
  s1 - s2  ->  {1, 2, 3}
  s2 - s1  ->  {8, 9, 7}

  s1.difference(s2)  ->  {1, 2, 3}
  s2.difference(s2)  ->  {8, 9, 7}
</code>
</pre>

##### 집합 관련 함수
* 값 1개 추가하기  ->  add(x)
<pre>
<code>
  s1 = set([1, 2, 3])
  s1.add(4)
  s1  ->  {1, 2, 3, 4}
</code>
</pre>

* 값 여러 개 추가하기  ->  update()
<pre>
<code>
  s1 = set([1, 2, 3])
  s1.update([4, 5, 6])
  s1  ->  {1, 2, 3, 4, 5, 6}
</code>
</pre>

* 특정 값 제거하기  ->  remove(x)
<pre>
<code>
  s1 = set([1, 2, 3])
  s1.remove(2)
  s1  ->  {1, 3}
</code>
</pre>
