02-5 딕셔너리 자료형
==================
* "이름"="홍길동" 과 같은 대응관계를 나타내는 자료형임.
* 딕셔너리(dictionary), 연관 배열(associative array), 해시(hash) 라고도 함.
* Key와 Value를 한 쌍으로 가지는 자료형임.
* 리스트나 튜플처럼 순차적으로 요솟값을 구하지 않고 Key를 통해 Value를 얻음.

##### 딕셔너리 만들기
<pre>
<code>
  {Key1: Value1, Key2: Value2, Key3: Value3, ...}

  dic = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
  a = {1: 'hi'}
  b = {'a': [1, 2, 3]}  ->  Value에 리스트도 넣을 수 있음.
</code>
</pre>

##### 딕셔너리 쌍 추가하기
<pre>
<code>
  a = {1: 'a'}
  a[2] = 'b'  ->  {2: 'b'} 쌍 추가
  a  ->  {1: 'a', 2: 'b'}
  a['name'] = 'pey'  ->  {'name': 'pey'} 쌍 추가
  a  ->  {1: 'a', 2: 'b', 'name': 'pey'}
  a[3] = [1, 2, 3]  ->  {3: [1, 2, 3]} 쌍 추가
  a  ->  {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}
</code>
</pre>

##### 딕셔너리 요소 삭제하기
<pre>
<code>
  del a[1]  ->  key가 1인 Key: Value 쌍 삭제
  a  ->  [2: 'b', 'name': 'pey', 3: [1, 2, 3]}
</code>
</pre>

##### 딕셔너리 사용
* 예를 들어 4명의 사람과 각각의 특기를 표현할 수 있는 방법으로 딕셔너리가 적절함.
<pre>
<code>
  {"김연아": "피겨스케이팅", "류현진": "야구", "손흥민": "축구", "귀도": "파이썬"}
</code>
</pre>

* 딕셔너리에서 Key를 사용해 Value 얻기
<pre>
<code>
  grade = {'pey': 10, 'julliet': 99}
  grade['pey']  ->  10
  grade['julliet']  ->  99
</code>
</pre>

##### 딕셔너리 만들 때 주의사항
* Key는 고유한 값이므로 중복되는 Key 값을 설정해 놓으면 하나를 제외한 나머지 것들이 모두 무시됨.
* Key에 리스트는 쓸 수 없음. 값이 변하면 안되기 때문임. (Value는 다 가능)
<pre>
<code>
  a = {1: 'a', 1: 'b'}
  a  ->  {1: 'b'}  ->  1: 'a' 쌍이 무시됨.
  b = {[1, 2]: 'hi'}  ->  error
</code>
</pre>

##### 딕셔너리 관련 함수
* Key 리스트 만들기  ->  keys()  ->  Key들을 모아 dict_keys 객체를 리턴함.
<pre>
<code>
  a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1110'}
  a.keys()  ->  dict_keys(['name', 'phone', 'birth'])
  
  for k in a.keys()  ->  리스트와 비슷하게 사용가능. 리스트 고유 함수는 사용 불가.
      print()

  list(a.keys())  ->  ['name', 'phone', 'birth']  ->  dict_keys 객체를 리스트로 변환
</code>
</pre>

* Value 리스트 만들기  ->  values()  ->  value들을 모아 dict_values 객체를 리턴함.
<pre>
<code>
  a.values()  ->  dict_values(['pey', '010-9999-1234', '1118'])
</code>
</pre>

* Key, Value 쌍 얻기  ->  items()  ->  Key와 Value의 쌍을 튜플로 묶은 값을 dict_item 객체로 리턴함.
<pre>
<code>
  a.items()  ->  dict_items([('name', 'pey'), ('phone', '010-9999-1234'), ('birth', '1118')])
</code>
</pre>

* Key, Value 쌍 모두 지우기  ->  clear()  ->  딕셔너리 안의 모든 요소 삭제.
<pre>
<code>
  a.clear()
  a  ->  {}
</code>
</pre>

* Key로 Value 얻기  ->  get(x)  -> x라는 Key에 대응하는 Value를 리턴함.
<pre>
<code>
  a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
  a.get('name')  ->  'pey'
  a.get('phone')  ->  '010-9999-1234'
</code>
</pre>

<pre>
<code>
  a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
  print(a.get('nokey'))  ->  None 출력
  print(a['nokey'])  ->  error
  print(a.get('nokey', 'foo')  ->  'foo'  ->  default 값으로 설정한 foo 출력
</code>
</pre>

* 해당 Key가 딕셔너리 않에 있는지 조사하기  ->  in
<pre>
<code>
  a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
  'name' in a  ->  True
  'email' in a  ->  False
</code>
</pre>
