04-1 함수
==================
* 입력값을 가지고 어떤 일을 수행한 후 그 결과물을 내어 놓는 것.
* 반복적으로 사용되는 가치 있는 부분을 한 뭉치로 묶어 어떤 입력값을 주었을 때 어떤 결괏값을 리턴해 줌.
* 프로그램을 기능 단위의 함수로 분리하여 흐름을 한눈에 볼 수 있음.

##### 파이썬 함수의 구조
* def는 함수를 만드는 예약어. 
<pre>
<code>
  def 함수_이름(매개변수):
      수행할_문장1
      수행할_문장2
</code>
</pre>

<pre>
<code>
  def add(a, b):
      return a + b

  a = 3
  b = 4
  c = add(a, b)
  print(c)  ->  7
</code>
</pre>

##### 매개변수와 인수
* 매개변수는 함수에 입력으로 전달된 값을 받는 변수.
* 인수는 함수를 호출할 때 전달하는 입력값
<pre>
<code>
  def add(a, b)  ->  a, b는 매개변수
      return a + b
  print(add(3, 4))  ->  3, 4는 인수
</code>
</pre>

##### 입력값과 리턴값에 따른 함수의 4가지 유형
* 일반적인 함수(입력값 O, 리턴값 O)
<pre>
<code>
  def 함수_이름(매개변수):
      수행할_문장
      return 리턴값

  리턴값을_받을_변수 = 함수_이름(입력_인수1, 입력_인수2, ...)

  def add(a, b)
      result = a + b
      return result
  a = add(3, 4)
</code>
</pre>

* 입력값이 없는 함수
<pre>
<code>
  def say():
      return 'Hi'
  a = say()
  print(a)  ->  Hi

  리턴값을_받을_변수 = 함수_이름()
</code>
</pre>

* 리턴값이 없는 함수
<pre>
<code>
  def add(a, b):
      print("%d, %d의 합은 %d입니다." % (a, b, a + b))
  add(3, 4)  ->  3, 4의 합은 7입니다.

  함수_이름(입력_인수1, 입력_인수2, ...)

  a = add(3, 4)
  print(a)  ->  None
</code>
</pre>

* 입력값도 리턴값도 없는 함수
<pre>
<code>
  def say():
      print('Hi')
  say()  ->  Hi

  함수_이름()
</code>
</pre>
